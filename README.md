# Pesach 2026 Shopping Checklist

Live app: https://alastairlivingston-sudo.github.io/pesach-2026/

Real-time shared checklist for Pesach 2026. Two users sync via JSONBin polling every 5s.
Three views: By Store, By Meal, Meal Plan. Tap any meal in Meal Plan to see its ingredients.
Single HTML file. No build step. GitHub Pages hosting.
